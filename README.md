## 🧠 Overview

Kemenarmy Trivia is a dynamic web application built to engage users through interactive quizzes and real-time feedback. The platform focuses on delivering a smooth, responsive experience with visually engaging UI elements and seamless state management.

This project demonstrates my ability to build scalable frontend systems with a strong emphasis on performance, usability, and clean architecture.

---

## ✨ Features

* 🎮 Interactive trivia gameplay
* ⚡ Real-time score tracking
* 📱 Fully responsive design (mobile-first)
* 🎨 Smooth animations using GSAP
* 🧩 Reusable and modular React components
* 🔄 Dynamic state management
* 📊 User input handling and validation

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Animations:** GSAP
* **State Management:** React Hooks
* **Other Tools:** Git, GitHub

---

## 🧱 Architecture & Approach

* Component-based architecture for scalability
* Separation of concerns (UI, logic, state)
* Optimized rendering for performance
* Clean and maintainable code structure

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/kemenarmy-trivia.git
cd kemenarmy-trivia
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
